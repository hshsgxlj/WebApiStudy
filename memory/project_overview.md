## 项目信息
- 项目名：WebApiStudy
- 类型：.NET Web API 学习项目

## 技术栈

- ASP.NET Core Web API
- SQL Server 2025 Enterprise Developer Edition（免费，开发/测试用）

- `Models/` — 数据模型和验证

- `Data/` — 数据层
